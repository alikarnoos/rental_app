دليل سريع:
1) ضع هذا المجلد داخل htdocs (XAMPP): C:\xampp\htdocs\rental_app
2) نفّذ ملف schema.sql في phpMyAdmin لإنشاء قاعدة البيانات والجداول.
3) عدّل config.php إذا كان اسم المستخدم أو كلمة المرور مختلفة.
4) شغّل Apache و MySQL من XAMPP ثم افتح: http://localhost/rental_app/index.php
5) المجلد uploads/properties سيحتوي على الملفات المرفوعة.
